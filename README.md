# ZoneSpider

空间说说爬虫，爬点赞人员并统计点赞次数，评论次数，id，名字

因为空间不把点赞人数显示完，所以只能算是半成品

# Dependences

request 2.23.0

demjson 2.24

lxml 4.5.0

xlutils 2.0.0

# Usage

把有效cookies放到cookies.txt里面，然后

> python Run.py
